using **Nexus API**

The mod for the Nexia CTS server.

### Requirements

- Java 21
- Gradle 8.7

### Build
- Clone the repository
  - `git clone https://github.com/nexia-cts/Nexia-Mod`
    - `git checkout 'main/dev'`
- Run `./gradlew build`
